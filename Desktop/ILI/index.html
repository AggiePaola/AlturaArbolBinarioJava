Somos la institución universitaria para todos y nuestro objetivo es integrar a la comunidad con el sector productivo para afrontar los desafios del futuro.

Este será tu perfil:

El ingeniero de software es competente en todas y cada una de las etapas que comprendel el cicli de vida de construccion de software como requerimientos, análisis, diseño, arquitectura, implementación, pruebas y puesta en marcha, así como en adminsitrar tiempos, recursos y procesos de una manera efectiva para la finalización exitosa de proyectos de software.

Sé un experto en:

  • Diseño y desarrollo web.

  • Diseño UX/UI.

  • Diseño y desarrollo de aplicaciones móviles.

  • Arquitectura de software

  • Desarrollo backEnd.

  • Desarrollo frontEnd.

  • Ingeniería de requerimientos.

  • Cloud Computing

  • Testing y aseguramiento de la calidad.

  • Metodologías ágiles de desarrollo.

  • Gestión de proyectos de software.

  • Integración de tecnologías.
